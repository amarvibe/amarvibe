Hi, I’m Amar — a front-end developer who creates clean, responsive web templates with the help of AI. I focus on simplicity, usability, and performance using HTML, CSS, JavaScript, and Tailwind CSS.

If you're a developer, student, or startup looking for quick, reliable web layouts, feel free to explore and use anything here.
